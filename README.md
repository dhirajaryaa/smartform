# AI-Powered Smart Form Auto Filler

## Overview

### Pain Point
In daily life, users visit numerous websites and fill out forms for login, signup, applications, surveys, etc. Entering the same data repeatedly is a repetitive and boring task. What if users could fill forms with one click using relevant user data?

### Solution
An AI-powered smart form auto filler Chrome extension that enables one-click filling of any form with relevant or random data, powered by AI.

**Deadline:** 4 Feb 2026

## Features
1. **Auto Fill Inputs**: Automatically detect and fill form fields with one click.
2. **Temporary Email**: Use disposable email addresses for email fields.
3. **Temporary Mobile Number**: Generate temporary phone numbers for mobile fields.
4. **AI-Powered Personalized Data**: Generate contextually relevant and personalized data instead of purely random values, using AI intelligence.

## Tech Stack
- **WXT Framework**: For building the Chrome extension efficiently.
- **TypeScript**: For type-safe JavaScript development.
- **Tailwind CSS**: For rapid and customizable styling.
- **Gemini AI API**: For AI-driven data generation and personalization.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/your-username/ai-form-filler.git
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Build the extension:
   ```
   npm run build
   ```
4. Load the extension in Chrome:
   - Go to `chrome://extensions/`.
   - Enable "Developer mode".
   - Click "Load unpacked" and select the `dist` folder.

## Usage
1. Install the extension as above.
2. Navigate to a webpage with a form.
3. Click the extension icon and select "Auto Fill" to populate the form with AI-generated data.
4. Customize settings for personalized vs. random data if needed.
